# 📅 오늘도 0% (Today 0%)

알바 스케줄 관리와 실시간 피로도 체크를 위한 웹 애플리케이션입니다.

## 🚀 프로젝트 소개
대학생 아르바이트생의 효율적인 일정 관리와 피로도 측정을 위해 제작되었습니다. 
바쁜 학업과 알바를 병행하며 지치지 않도록 자신의 상태를 기록하고 확인합니다.

## 🛠 주요 기능
- **일정 관리**: 알바 근무 시간을 입력하고 관리합니다.
- **피로도 체크**: 현재 나의 피로도를 수치화하여 기록합니다.
- **간편한 UI**: 복잡한 설정 없이 바로 일정을 확인할 수 있습니다.

## 💻 사용 기술
- HTML5, TypeScript
- Bun (Runtime)
- Git / GitHub

## ⚙️ 실행 방법
1. 프로젝트 폴더로 이동합니다.
   ```bash
   cd today_zero_percent
   ```
2. 필요한 라이브러리를 설치합니다.
   ```bash
   npm install --legacy-peer-deps
   ```
3. 프로젝트를 실행합니다.
   ```bash
   npm start
   ```
